
# 🚀 Custom PHP MVC Blog System

A fully-featured, secure, and dynamic Blogging System built from scratch using native **PHP (MVC Architecture)** and **MySQL**. 

## ✨ Features
- **User Authentication:** Secure Login and Registration system.
- **Admin Dashboard:** Full CRUD operations for managing blog posts (Create, Read, Update, Delete).
- **Interactive Comments:** Dynamic comment section for readers on each post.
- **Clean Architecture:** Built using the Model-View-Controller (MVC) design pattern for high maintainability.
- **Database Management:** Optimized MySQL relational database.

## 🛠️ Tech Stack
- **Backend:** PHP (OOP, MVC)
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
